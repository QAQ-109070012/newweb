 餐點介紹

雙層牛肉經典漢堡 ( NT$250 ) :
       由生菜、洋蔥、番茄、酸黃瓜、起司與雙層手工牛肉排組成，是最經典的美味。大口咬下，美味十足 !

羅勒炸雞 ( 套餐附贈 ) :
       酥脆的外皮，雞肉鮮而不柴，保有十足的肉汁。

提拉米蘇 ( NT$100 )
       濃郁的起司伴隨著酥脆的餅乾，內層則是軟綿滑嫩的驚喜，喜歡甜食的你一定不能錯過 !
