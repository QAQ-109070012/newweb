 菜單

 餐點介紹

燻雞起司三明治 ( NT$150 ) :
       包含三明治、薯條與炒蛋。香噴噴的土司加上豐富的餡料，一口咬下去能嘗到滿滿的美味，再搭配上酥脆的薯條和鮮嫩的炒蛋，令人感到幸福。

特製布丁與濃湯 ( 套餐附贈 ) :
       滑嫩可口的布丁，加上可愛的貓掌圖案，讓人讚不絕口，以及濃郁的巧達濃湯，裡面包含了滿滿的用料，還有一塊可愛的貓型餅乾 !

咖哩雞腿飯( NT$200 )
       可愛的造型淋上濃郁微辣的咖哩醬，一湯匙挖起來是滿滿的配料與大大的雞腿塊，帶給你不一樣的美味。
