 菜單

 餐點介紹

天使蝦水波蛋 ( NT$370 ) :
       這間餐廳的主打菜，鬆軟的麵包搭上濃郁的蛋汁與起司醬，配合著新鮮的天使蝦，讓人的口中滿是幸福感。

炸雞塊與炒薯塊 ( 套餐附贈 ) :
       酥脆多汁的大雞塊與清香的炒薯塊，搭在一起是另類的享受。

松露野菇燉飯 ( NT$280 )
       水波蛋的蛋黃與松露襯托出這道料理的美味，滿滿的配料與濃郁的口感使人讚不絕口。
